## Installation

```bash
yarn install
```

## Running

```bash
# development mode
yarn dev
# before running in production mode
yarn build
# production mode
yarn start
```

# Features
- Apollo GraphQL
- React-Hook-Form (Form Validation)
- Redux (UserInfo and Dashboard API results)
- SWR (Data fetching)
- JSON-web-token
